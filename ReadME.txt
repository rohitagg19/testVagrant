To Execute follow the steps.
1. Import the project in Eclipse.
2. Once Imported successfully just perform right click on testng.xml.
3. Then select Run As--> TestNG suite.
4. After execution report will be created in the project directory.


Note:- 
	1. Just have a look at the ConfigFile.xml to provide configuration details.
	2. For Executing using maven please use "mvn test" command. if it fails, try "mvn compile test"
